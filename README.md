## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev'

 ExamVerse AI

 🚀 Overview

ExamVerse AI is an AI-powered exam preparation platform designed to help students prepare smarter. It provides personalized learning resources, previous year papers, mock tests, AI-powered recommendations, and exam information in one place.

✨ Features

* 🔐 Secure User Authentication
* 🤖 AI-powered Exam Recommendations
* 📅 Exam Calendar
* 📖 Previous Year Question Papers
* 📝 Mock Tests & Practice Quizzes
* 📊 Student Dashboard
* ⚖️ Law Hub for Law Students

 ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/YOUR_USERNAME/ExamVerse--AI.git
```

### Navigate to Project

```bash
cd ExamVerse--AI
```

### Install Dependencies

```bash
npm install
```

### Add Environment Variables

Create a `.env.local` file

```env
GEMINI_API_KEY=YOUR_API_KEY
```

### Run the Project

```bash
npm run dev
```
